# Indian EV Sales Dashboard (2015–2024)

## Overview

This Power BI dashboard analyzes India's Electric Vehicle sales between 2015 and 2024 using data from Kaggle.

## Dashboard Highlights

- Total EV Sales
- Total Manufacturers
- Average Sales per Manufacturer
- Top 10 EV Manufacturers by Sales
- EV Sales Trend (2015–2024)
- Vehicle Category Distribution
- Top 10 States by Operational Charging Stations

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dataset

Source: Kaggle (Vahan EV Dataset)

## Insights

- 2W vehicles dominate EV sales.
- Maharashtra has the largest charging infrastructure.
- OLA Electric is the highest-selling EV manufacturer.
- EV adoption increased rapidly after 2021.
